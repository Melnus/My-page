### これはできません・しません

* アニメやゲームのキャラクターなどの依頼者に権利がないもの
* MMDへの作成・コンバート・編集（必要があればご自身でしていただきます）
* 作成元がわからないモデル
* 大がかりな機材が必要な作業(全身トラッキングや肉体をそのまま３Dモデルに変換するものなど)



### 秘密保持・契約に関して

* 依頼品に関する作業中の情報公開は一切しません。納品後、必要があれば(または依頼者からの許可があれば)行います。
* 商品のやり取りに関してはBoothを行います。
* 契約に関してはDiscord上にてお知らせいただいた際に、すみやかに作成物等を破棄します。
* 商用利用に関して、依頼での実作業分以外の報酬を受け取りません。(二次利用に関して関知しません)
* 依頼品に関する作業に関する情報はDiscordでのログを記録とします。必要があれば別途文書にします。

### 著作権、著作財産権、著作者人格権について

* デザイン持ち込みでの作成に関しては、元のデザインがそのデザイン作成者、モデル作成に関しては私の名前とさせていただきます。
* 著作権に関しては**クリエイティブコモンズCCBY**に準拠します\
  [https://creativecommons.org/licenses/by/4.0/deed.ja](https://creativecommons.org/licenses/by/4.0/deed.ja)
* 著作者人格権の譲渡はしません（作った人の名前はMelnusです）
* 著作財産権の譲渡はします（私用・商用利用可）
* 二次利用についてOK

### 依頼の受付開始・停止について

取引額に応じ停止する場合があります。停止の場合は現在受けている分の完了でもって停止します。

### 依頼者側から欲しいもの

* イメージ図、参考図
* 作成に関する細かい指定

* ほしい表情のリスト
* こだわりの部分、これはいじってほしくない部分\
  作成したモデルの動作に無理がある場合など、変えなければならない場合がありますので必要です
* 納品物\
